# Netflix Lite :

A lightweight movie recommendation system inspired by Netflix.

# Features
- Content-based movie recommendations
- Item-item collaborative filtering
- Cosine similarity for similarity calculation
- Flask-based web interface

# Tech Stack
- Python
- Flask
- Pandas
- Scikit-learn

# How It Works
1. Movie data is converted into feature vectors
2. Cosine similarity is used to find similar movies
3. Users get recommendations based on selected movie
